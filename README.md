## How to use
1. Install the Custom Joomla node if not already installed ([Joomreem Content](https://github.com/mabdelaziz77/n8n-nodes-joomreem-content)).
2. Import `joomla_trash_cleaner_workflow.json` into n8n.
3. Update the credentials node.
4. Set the `Specify Categories to Delete` in the "Set" node.
5. Click on "Execute workflow" to start the automation.
